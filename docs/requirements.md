# 要件定義

## プロジェクト概要

RubyKaigi 2026 向けのブラウザ完結型クイズアプリ。Ruby のメソッドチェーンを題材にした「処方箋クイズ」。

## 目的

- RubyKaigi 2026 の参加者・来場者が Ruby のメソッドチェーンを楽しみながら学べる

## 機能要件

### 必須（Must）

- [ ] 処方箋風メソッドチェーンスコアクイズを出題し、ユーザーが回答できる
- [ ] Ruby コードを WebAssembly（`@ruby/wasm-wasi`）でブラウザ上で実行し、正誤を判定する
- [ ] スコア・結果画面を表示する。レベル感はユーザーが体験してみて「まだまだいけそう！」と思うような4段階ぐらいを用意
- [ ] 結果画面には、スコアをSNS（X, twitter）でシェアできるボタンを用意
- [ ] 結果画面に オンライン診療で利用できる クーポンコード を提示する
- [ ] クーポンコードを利用できるオンライン診療のURLを提示する

### 推奨（Should）

- [ ] クイズを回答して、もう一度トライしやすいようにUIを工夫して
- [ ] モバイルでも操作しやすいように工夫して

### あれば嬉しい（Nice to have）

- [ ] レベル感が表示された際に、簡単なイラストを提示して
- [ ] 回答した際に、すぐに結果が出るのではなく、演出を取り入れて

## 非機能要件

- ブラウザのみで動作する（サーバー不要）
- モバイル・タブレット・デスクトップに対応するレスポンシブ対応
- GitHub Pages でホスティング可能な静的サイト
- 日本語・英語 UI（切り替え可能）
