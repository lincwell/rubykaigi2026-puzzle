# UI 設計

## デザインコンセプト

「処方箋」をモチーフにした医療・薬局風のビジュアル。Ruby のコードを「薬の処方」に見立てて出題する。

## フォント

`index.html` で読み込み済み:

| 用途 | フォント |
|------|----------|
| 見出し・本文（日本語） | Noto Serif JP / Noto Sans JP |
| コード表示 | JetBrains Mono |

## カラーパレット

| 役割 | カラー | 備考 |
|------|--------|------|
| プライマリ | #00b9f0 | 明るく爽やかなシアンブルー |
| セカンダリ | #6b7280 | |
| 正解 | #22c55e | 見やすく前向きなグリーン |
| 不正解 | #ef4444 | はっきり伝わる赤 |
| 背景 | #ffffff | |

## コンポーネント一覧

| コンポーネント | 概要 |
|--------------|------|
| TitleScreen | タイトル画面全体 |
| QuizScreen | クイズ出題画面 |
| FeedbackScreen | 回答フィードバック画面 |
| ResultScreen | 結果画面 |
| CodeBlock | Ruby コード表示ブロック |

## レスポンシブ対応

- ブレークポイント: Tailwind CSS v4 のデフォルト準拠
- モバイルファーストで設計
