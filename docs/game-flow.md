# ゲームフロー・画面遷移

## 画面一覧

| 画面ID | 名称 | 概要 |
|--------|------|------|
| S01 | タイトル・クイズ画面 | タイトル・問題文・入力欄・実行ボタンを表示 |
| S02 | 結果画面 | 回答終了後のスコアと診断結果の振り返り |

## 画面遷移図

```
S01 タイトル・クイズ画面
     └─（回答終了）→ S02 結果画面
                    └─[もう一度]→ S01 タイトル
```

## 各画面の詳細

### S01 タイトル・クイズ画面

- タイトルブロック表示要素:
  - 見出し1: おくすり
  - 見出し2: あなたの「Ruby力」を診断します
  - 説明文: "Lincwell"を起点に、最大の整数を返すメソッドチェーンを処方してください。

- クイズブロック表示要素:
  - 患者: "Lincwell"
  - 処方: chars, bytes, methods, class, to_s, inspect, join, size, length, sum (最大4列まで)
  - 用法: 上記メソッドをチェーンにて投与すること
  - 用量: 各1回まで
- 遷移: 投与ボタン押下（Ruby WASM で実行して判定） → S02


### S02 結果画面

- 回答表示要素:
  - メソッドごとに出力する
  - エラーなら残念。リトライしてボタンを用意。遷移: もう一度 → S01
  - エラーによって、ヒントも提示する

- 結果表示要素:
  - スコア（8, 20 etc.）
  - レベル感を表示。4段階ぐらい。
  - SNSのシェアボタンを用意
  - オンライン診療で利用できるクーポンコードを提示する
  - オンライン診療のURLを提示
  - もう一度ボタン
- 遷移: もう一度 → S01

### S01 & S02 固定共通要素

- ヘッダー:
  - Ruby WASM で実行する Ruby の version を表示

- フッター:
  - ロゴの svg ファイルを配置
